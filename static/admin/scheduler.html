      📐 One-Time Canva Template Setup Guide (read once, skip forever after)
      
      
        
      Create one Canva template at 1000×1500px with named text layers. Every pin you make is just: open template → paste text into layers → swap background image → download. Under 2 minutes per pin once set up.
      
        
      
          
      Step 1
      Create the canvas
      New design → Custom size → 1000 × 1500 px. Save as "WGD Pinterest Template". Never edit the master directly — always duplicate first.
      
          
      Step 2
      Add background image layer
      Full-canvas image placeholder at bottom layer. Use a dark overlay (60–70% opacity) on top so text is always readable.
      
          
      Step 3
      Create exactly 3 text layers
      • HEADLINE — bold, large (48–64pt). Top 60% of canvas.
      • SUBLINE — medium weight (24–30pt). Below headline.
      • URL WATERMARK — small (16pt), bottom. wowglamdecor.com
      
          
      Step 4
      Lock the URL watermark
      Right-click the URL layer → Lock. Your URL stays on every pin automatically.
      
          
      Step 5
      Per-pin workflow
      Duplicate template → swap background → paste HEADLINE → paste SUBLINE → download as JPG, quality 80% → upload to Pinterest.
